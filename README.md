# 🏄 Beach Surfer

A casual surfing game built with Python and Pygame. Ride the wave as long as you can without wiping out!

## Gameplay

Control the surfer with your mouse to stay inside the moving wave lane. Every frame you stay on the wave you score points. Stray too far from the center and your balance meter drains — hit zero and it's a **WIPEOUT!**

## Controls

| Input                 | Action                  |
| --------------------- | ----------------------- |
| Mouse (move up/down)  | Move the surfer         |
| Click / Enter / Space | Restart after game over |
| Close window          | Quit                    |

## Requirements

- Python 3.8+
- Pygame

```bash
pip install pygame
```

## Running the game

```bash
python main.py
```

## Features

- Procedurally animated wave path using layered sine functions
- Beach scene with sky gradient, sun, sea, sand, and palm trees
- Foam particle system
- Balance energy bar with visual damage feedback
- Score counter
- Game over / restart screen
